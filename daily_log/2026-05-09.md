## Landed today

**CX-74 (fde65ed, PR #117 → submain): Audit Cranelift CLI exit-code propagation.** CONFIRMED. This is the highest-impact merge of the day. Six silent `return;` paths in `main.rs` — where backend errors would exit 0 and the differential harness would misclassify them — are now replaced with structured `BackendError { message, exit_code }` propagation. The `Backend` trait gained a `BackendError` return type so exit codes flow from JIT to CLI without string parsing. JIT errors map to 127 (unsupported construct, codegen gap, missing main, IR lowering failure) or 126 (runtime panic). Three new exit-code tests added. This directly unblocks the differential harness from producing false SKIPs via exit-0+stderr; the canonical SKIP signal is now exit 127.

Updates to IR printer, validator, and JIT skeleton across 6 files. This was originally CX-48 but rebased onto current submain via CX-73.

## Decisions and direction

**Exit-code propagation formalized as architecture.** CX-74 establishes a contract: exit 127 = JIT can't handle this program (SKIP), exit 126 = runtime panic, exit N = Cx program returned N. The `BackendError` struct in the `Backend` trait makes this the canonical interface between backends and CLI. This was driven by CX-69's parity harness misclassifying fixtures when backends exited 0 on errors.

**CodeRabbit integrated as automated reviewer.** CX-67 adds `.coderabbit.yaml` with per-directory review instructions. CodeRabbit is now actively reviewing PRs (CX-70 and CX-71 are direct responses to its feedback). This is the first automated review infrastructure in the project — co-authored-by lines from CodeRabbit appear in several commits today.

**JIT harness now has a 30-second timeout.** CX-71 adds `wait-timeout` as a dev-dependency so hanging JIT subprocesses get killed and classified as PARITY_FAIL instead of blocking CI indefinitely.

**Assert/assert_eq leave the builtin gate.** CX-48/CX-73 remove these from `is_cx_builtin()`, making them the first builtins to graduate from runtime intrinsic dispatch to native Cranelift code generation via a Branch+Trap CFG pattern.
